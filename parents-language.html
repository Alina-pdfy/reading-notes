推荐序

孩子出生后的头三年，大脑极具可塑性，真正能拉开孩子之间差距的，不是家庭有没有钱、上了多少课外班，而是从出生起，父母有没有用丰富、积极的语言去陪伴他。所谓的“赢在起跑线”，本质上是指孩子从一开始就浸泡在优质的语言环境里。父母的语言，远比任何昂贵的早教班更能深远地影响孩子。

📌 人类的大脑是唯一在出生后还具有可塑性的器官，尤其是在三岁前。三岁前，大脑会发育到成熟期的80%左右，这三年里，使大脑建立神经连接的条件是什么呢？是语言，具体来说，是父母的语言。

📌 “赢在起跑线”上的孩子，并不是他们家境富有，或者报了多少课外班，而是他们一出生，就能够在语言丰富、积极正面的环境中熏陶。

📌 父母语言对孩子的意义，可能比任何昂贵的早教班都更加深远。

译者序

译者曾困惑，除了生命还能给孩子什么。这本书让她认识到，婴幼儿期是神经发育最快的阶段，潜能开发越早越充分，这就是所谓的”递减规律”。她最终意识到，真正决定孩子聪慧与否的，不是天赋，而是父母是否善于沟通。早期良好的语言环境，既促进智力发展，也稳固亲子关系。

📌 初为人母、毫无育儿经验的我如获至宝，仅仅粗略翻看一遍后，父母对儿童进行早期语言教育的内容就印刻在我的头脑中。

📌 儿童的潜在能力遵循着一种递减规律，即生下来具有100分潜在能力的儿童如果一出生就进行理想的教育，就可以成为具有100分能力的人；若从5岁开始教育，即使是理想的教育，也只能成为具有80分能力的人；若从10岁开始教育，就只能成为具有60分能力的人。

📌 这个世界上没有那么多天生聪颖的孩子，更多的聪慧源自其善于沟通的父母。

第一章 缘分：儿童人工耳蜗医生眼中的大脑发育

决定孩子学业表现的，不是家庭经济条件，而是早期的语言环境。听力正常却缺少语言互动的孩子，本质上与从未接触过语言符号的聋儿处境相似。丰富积极的语言输入，才是大脑发育真正的催化剂。

📌 父母的社会经济地位并不影响孩子学业的好坏，然而父母与孩子交谈中使用的语言，也就是说早期的语言环境直接决定了孩子日后的社会表现。

📌 听力正常但语言环境不佳的孩子，与天生耳聋且未接收大量语言符号的孩子无异。

我的故事

达娜·萨斯金德是一位儿童人工耳蜗植入外科医生。神经外科手术中目睹大脑奥秘的震撼，促使她转而选择耳朵作为研究大脑的切入点。本节梳理了她的职业轨迹，内容涉及她对大脑的敬畏、人工耳蜗工作原理及新生儿听力筛查的重要性。

📌 一名负责儿童人工耳蜗植入的外科医生写了一本关于父母语言影响的书，我个人觉得这并没有什么好奇怪的。

📌 看着手术台上的病人，我突然想到自己脑袋里怦怦跳动的大脑。难道就是这一团黏糊糊的东西在掌管着我们的思想和行为？

📌 之前在手术室不太好的经历让我想换一种方式来和大脑打交道。耳朵，就是其中一种我选择的方式

人工耳蜗植入

手术成功不等于孩子就能康复。三岁前大脑已完成约85%的发育，而这个过程高度依赖语言环境。即便在最佳时机植入人工耳蜗，若术后缺乏丰富的语言互动，康复效果仍可能不佳。语言环境与手术本身同等重要。

📌 孩子长到三岁时，脑部发育已完成85%，脑神经细胞多达数千亿个，此时的大脑正为思考和学习做准备

📌 科学研究证实，脑部发育与幼儿的语言环境息息相关。不是说大脑在孩子三岁以后就停止发育了，而是三岁以前是大脑发育的黄金期。

📌 然而，后来我了解到，即使在最佳时间植入了人工耳蜗，也有可能因为一些原因导致康复效果不佳。

进展缓慢让我受益

在芝加哥南部行医期间，作者因接诊量小得以细致观察每位患儿。她发现，初始条件相近的几个孩子，术后表现却截然不同。这一现象促使她反思：一味依赖大数据可能掩盖关键细节。孩子潜力本身并无上限，能否释放取决于是否获得适当的机会。

📌 对没有机会学习的孩子来说，他们的潜力是无穷的。

📌 更让人费解的是，起初情况很相似的几个小孩，做出的反应却截然不同。在寻找答案的过程中，我很偶然地走进了先天性聋儿的世界。

📌 但我慢慢地意识到若不顾及自己在研究中的个人感受，一味追求所谓的大数据，反而会让我们忽略一些重要的发现。

扎克和米歇尔

扎克与米歇尔构成鲜明对照。扎克父母悉心投入，让扎克感受喉部振动，将恢复听觉之日定为”听觉生日”，扎克三岁便学会说话。米歇尔母亲虽深爱女儿，却甚少进行语言互动，米歇尔能听见却无法理解含义，原有光彩逐渐褪去。这一对比让作者意识到，语言环境与人工耳蜗手术本身一样，是决定康复效果的关键变量。

📌 扎克的父母决定把他恢复听觉的那一天，定为他的”听觉生日”。

📌 米歇尔丰富的语言环境就像氧气。当你拥有它的时候，你觉得理所当然，当你没有它的时候，你才会意识到它是如此重要。

📌 我开始慢慢思考耳蜗植入以外的因素，因为这些造成米歇尔和扎克学习能力有差别的因素大同小异，关系到一个人的潜力。

语言环境与学习水平

三年级时，扎克阅读能力完全达标，米歇尔身上的光彩却已彻底消失，语言环境差异最终体现为学习水平的巨大鸿沟。作者由此反思：社会经济地位不影响孩子学说话的能力，医生的责任不限于手术成功，更须确保患者获得真正良好的康复。

📌 孩子在小学三年级时的阅读水平就可以反映出他未来的学习水平。扎克的阅读水平完全达到了三年级水平，年龄与水平相符。

📌 她不再是我初见的那个蹒跚学步、充满活力的米歇尔，她的光辉似乎完全褪去。

📌 尽管他们的家庭背景迥异，但是社会地位和经济条件绝不会影响一个孩子学习说话。

左手科学，右手责任

手术顺利不等于真正成功。家庭社会经济地位虽不阻碍为孩子提供关爱，却深刻影响其受教育水平与康复效果。若要真正帮助患者，医生必须走出手术室，直面决定健康的社会因素，而这项工作远较手术本身艰难。

📌 手术的成功或许不意味着真正的成功。

📌 一个家庭的社会地位和经济条件会严重影响孩子受教育的水平、健康状况和疾病康复效果。

📌 作为外科医生，为社会问题去寻找解决办法意味着我必须走出熟悉的医院和手术室，这个难度对我来说不亚于登上月球。

贝蒂·哈特和托德·里斯利

哈特与里斯利的研究推翻了学习好坏取决于天生智力的传统观念。研究发现，贫困家庭与富裕家庭之间的本质差异在于语言环境，词汇量的差距直接导致学业差距。核心结论是：社会经济地位并非关键因素，决定性的因素在于父母与孩子交谈时所使用的语言。孩子三岁前所听到的词汇数量与质量，直接影响其日后的学业表现。

📌 要知道在那个年代，人们的传统思维是：若一个人学习好，那肯定是因为他脑子够用；若学习不好，那肯定是他脑子不够聪明，没有别的解释。

📌 贫困家庭的语言环境和富裕家庭的语言环境是有差别的，这将直接导致孩子今后在学业上的差别。贫困家庭的孩子接收的词汇量远远低于富裕家庭的孩子。

📌 父母的社会经济地位并不影响孩子学业的好坏，然而，父母与孩子交谈中使用的语言才是最最关键的影响因素。也就是说早期语言环境的好坏直接决定着孩子今后在学业中的表现。

梦开始的地方

聋儿案例再次证实了这一规律：语言环境决定学习成效。赋予听力不等于获得理解能力，脱离语言土壤的植入终究徒劳。正是这份醒悟，促成本书诞生，以帮助更多家庭激活孩子的潜能。

📌 这些聋儿中，语言环境好的同样学得好，语言环境不佳的孩子学习情况也同样不容乐观

📌 尽管人工耳蜗的确可以赋予孩子们听力，但如果脱离了语言环境，即便安装了也是徒劳，孩子们也是不可能学好语言的。

📌 这就是我撰写这本书的初衷。

第二章 3000万词汇：父母语言的力量

早期语言环境即决定了孩子日后的学业表现，交谈越多，词汇增长越快，智商测试成绩也越高。“刺柏花园”项目虽未奏效，却揭示了一个关键事实：短期词汇集训无法消除差距，真正需要改变的，是人们对学习能力差异根源的根本认知。

📌 家长与孩子的对话越多，孩子词汇量会增长得越快，孩子日后测试出来的智商会越高。

📌 到进入幼儿园的时候，那些积极的效果消失了，这些孩子和没有提前参加过入园词汇集训的孩子居然没什么差别。

📌 这次失败的研究，至少让他们知道了人们对于”孩子们学习能力有差异”这一点的认识是有问题的。转变这一传统观念势在必行。

儿童清醒时间应该做什么

哈特与里斯利将目光投向一个尚属空白的领域：婴幼儿每周约110小时清醒时间的具体活动内容，及其对最终学业成就的影响。当时有关婴幼儿日常生活的可靠学术资料极为匮乏，令寻找确凿答案的路径困难重重。

📌 于是，他们提出了两个问题：1．婴幼儿和儿童一个星期内醒着的那110个小时到底做了些什么？2．它们对孩子们最终的学习成绩有多大的影响？

📌 因为你必须面临这样一个残酷的现实：关于婴幼儿日常生活的文献资料极其匮乏。也许有为数不多的几篇论文，但也难以得出令人信服的结论。

早期语言环境的重要研究

乔姆斯基的理论认为，大脑内置了通用的语言装置，出生后两三年由环境设定参数，这一学说令遗传决定论广为人知。哈特与里斯利不满足于此，坚持用实证数据检验。托德·里斯利是发展心理学的重要奠基人，贝蒂·哈特以对细节的极致追求著称，二人的合作为后来的认知转向奠定了根基。

📌 乔姆斯基认为，大脑中有专门的语言装置，我们每个人从一出生就具有掌握人类一切语言的能力，在两三年间由外部环境给语言装置设定特定的参数，于是人就具体掌握了一门语言，这就足以解释为什么在幼儿时期学习语言特别快。

📌 乔姆斯基的理论获得人们的广泛接受，这就意味着遗传的重要性被普遍接受。

📌 她追求细节，从不放过任何一个精确的数据。如此求真务实的精神让她的梦想一步步成为现实。

3000万词汇差距初现

研究发现，高社会经济地位家庭子女每小时平均听到2000个单词，而贫困家庭子女仅听到600个；前者每小时约收到40个口语肯定，后者仅4个。影响学习能力的决定性因素并非社会经济水平、种族、性别或出生顺序，而是早期语言环境。

📌 在一个小时内，高社会经济地位家庭的孩子平均听到的单词数量是2000个。贫困家庭的孩子听到的单词仅仅只有600个。

📌 早期的语言环境才是影响孩子最终学习能力的关键，即，父母应该对孩子说多少话，应该怎么和孩子说话。孩子感受到父母的语言越多，语言能力会越好，与父母受教育程度和社会经济地位没有太大关联。

📌 儿童早期语言学习效果的确会影响其入学后的学业成绩，也就是说，错过对儿童早期语言能力的培养就可能造成孩子以后在学业上和其他同学有差距。

研究结果可信吗

研究虽因样本量有限受质疑，数据却揭示了一条深层规律：语言数量的增加，会自然推动语言质量的提升。日常自发闲谈远比事务性指令对话更具价值。负面语言的差距同样显著——贫困家庭孩子每小时听到的否定性话语，是脑力劳动者家庭孩子的两倍有余。弗纳尔德的研究进一步证实：在两岁阶段，不同语言环境中的幼儿在词汇量和语言处理速度上，已相差整整六个月。

📌 里斯利说：“我们不必要求父母对孩子说不一样的话，我们只需要帮助父母多说话。“而剩下的就不需要父母操心了。

📌 社会经济地位高的家庭会训斥孩子，但频率远远低于社会经济地位低的群体。贫困家庭的孩子每小时听到的负面语言是脑力劳动者家庭孩子的两倍多。

📌 安妮·弗纳尔德也发现了哈特与里斯利曾发现的社会经济关系。在她的研究中，社会经济地位低的家庭的两岁幼儿与社会经济地位高的家庭中的同龄孩子在词汇量和语言处理技能上存在着整整六个月的差距。

第三章 神经可塑性：早期语言环境的决定性影响

宝宝从出生起，大脑便以每秒700至1000条新连接的速度发育，到四岁时大部分发育已完成。早期语言环境贫乏的孩子，语言处理速度与效率都明显更低。更需警惕的是，早期不良环境对大脑的影响，后期即使再努力介入也难以完全弥补——三岁前这几年，是大脑发育不可错过的黄金窗口。

📌 宝宝一出生，每1秒都会产生700～1000条额外的类神经连接，其带来的让人难以置信、错综复杂的大脑回路便是大脑的构造，影响孩子所有的大脑功能，包括记忆力、情感、动手能力，当然还有语言表达能力。

📌 在四岁的时候，我们生理大脑的很大一部分就已经完成生长发育了。

📌 他们的介入非常有效，但是如果早期语言环境不太理想，从而对大脑造成了一定的影响，那么是无法改善的。

儿童大脑的后天发育

大脑发育不仅需要物质营养，更依赖“社会营养”。基因只是蓝图，真正决定建筑质量的是早期经历，而持续的压力激素甚至可能永久性改变大脑结构。在所有影响中，亲子关系与语言环境最为核心——词汇能产生多大的积极影响，取决于回应的及时性与温和程度。

📌 传统意义上，牛奶被视为营养所需，能够满足生存与成长所需。我们直到现在才明白，除了身体成长所需食物之外，我们也同样需要最佳的社会营养才能有效保证智力发育。这两种需求都绝对地依赖于照顾者。

📌 在大脑发育过程中，最重要的因素在于宝宝跟父母之间的关系，其中包括语言环境。简短的”爸爸爱他的心肝宝贝”这样的话语片段，对于一个刚刚开始集中注意力的孩子来说真的很重要吗？是的，真的非常非常重要。

📌 尽管有很多话语，但是它们对大脑的积极影响依赖于响应能力和温和度。

静面实验（面无表情实验）

大脑发育的核心，不在于物质条件，而在于孩子能否与一位充满爱意、积极回应的成年人持续互动。冷漠、缺乏表情甚至充满敌意的环境造成的伤害，绝非事后简单拥抱所能弥补。基因与天赋并非固定不变的代码，而是在每日的互动中被持续塑造。

📌 孩子一生的学习、行为以及健康状况都建立在这种与父母积极的、相互回应的、礼尚往来的基础之上。从本质上来说，大脑发育的核心在于宝宝与一位有爱心，能做出积极回应的成年人之间的互动。

📌 长期生活在”面无表情”或更糟糕、充满愤怒或是怀揣敌意的环境中，这肯定是无法在短短的几秒钟时间内就能用一个拥抱弥补的。

父母应尽量增强儿童脑回路构建

大脑约有一千亿个神经元，缺乏连接便如同孤立的电话亭，毫无意义。三岁前，大脑每秒都在快速建立新连接，随后又系统性地修剪掉那些不常用的通路。这一过程既是机遇也是风险：一旦错过关键语言期，孩子即使能够听见声音，也已无法真正理解其含义。时机，远比任何后期的补救都更为关键。

📌 我们每个人与生俱来就有一千亿条神经元潜能，能转化成诸多不同的潜力。但是，如果没有关键性的类神经连接，那一千亿条神经元也就毫无意义，就像没有连接线的独立电话亭一样。

📌 不过杰克·宋可夫(Jack Shonkoff)博士也说：“这是一个充满机遇与弱势的双重时机。”

📌 现在她终于明白”听得到”不等于”听得懂”了。

大脑发育的关键时期

视觉发育的逻辑同样适用于大脑整体：想看清晰，先要有东西可看。大脑严格遵循”用进废退”，凡是不用的神经连接都会被系统性地修剪掉。休伯尔与威塞尔因揭开这一机制而获诺贝尔奖，其实验有力证明——早期感官刺激，直接决定了发育的质量与走向。

📌 但是，跟语言一样，视敏度对环境也有依赖性。简言之，为了看清事物，宝宝需要有能够看的东西。

📌 大脑进入其”用进废退”模式，并开始修剪过程，淘汰掉那些未使用或是脆弱的类神经连接。

时机决定一切

大脑发育自有其不可逾越的顺序：基础能力不扎实，更复杂的技能就无从谈起。生命头三年的语言积累，为日后的社交、情感和认知发展奠定了根基。无论听力是否正常，成长中若严重缺乏词汇，留下的损伤将是永久性的。

📌 错失良机，对于一项技能而言，是非常严重的，因为在大脑的发育过程中，基本的能力是获得一项更加复杂的技能的前提，每项技能都犹如修建一幢大楼，必须修建好了一个街区才能更好地修建下一个。

📌 如果一个小孩子的一生缺乏词汇，不管他听力是否受损，将来他的口头表达能力或手语能力，都会受到永久性的不利影响。

听、读与学习能力

三年级是学业路上的一道分水岭。语言能力强的孩子会因积累而愈发领先，语言能力弱的孩子则因基础薄弱而愈发吃力，差距随时间持续扩大。语言的输入数量与品质，正是驱动大脑发育的核心动力——每一次习得，都在为下一步的真正掌握铺路。

📌 三年级是关键性的一学年，儿童会在这一年中将书页上简单的字母如连珠炮一般脱口而出，从而形成自己的观点，并从这些单词中积累知识。

📌 心理学家吉斯·斯坦诺维奇(Keith Stanovich)将这叫作”马太效应”，因为它是根据《马太福音》中”凡有的，还要加给他，叫他有余。凡没有的，连他所有的，也要夺去”而命名的。

📌 语言的输入、数量和质量都是大脑发育的主要促进因素。

我们都曾是计算天才

刚刚来到这个世界的婴儿，天生就能分辨人类所有语言的细微差别，是真正的“世界公民”。大脑会先将捕捉到的语音模式进行统计归类，再逐步解码成有意义的语言。讽刺的是，成年人穷尽心力学习外语的方式，效率远不及这些尚不会说话的婴儿。

📌 在我们理解或者说出某个词汇之前，我们的大脑不得不进行”句法分析”，即拆分音素后再将它们拼接在一起并创造出词汇。这是早期大脑学习本地语言过程中的重要组成部分。

📌 有趣的是，即便是成年人也比不上新生儿的天赋。

为什么年纪越大越难学习新语言

七个月大的婴儿尚能分辨所有语言的细微语音差别，三个月后，这项能力便迅速消失。大脑会主动修剪那些未被使用的语音识别通路，仅保留与母语相关的部分。父母那种夸张、缓慢的”儿向言语”，恰恰是在帮助婴儿提取语音信息，引导其将注意力集中于即将使用的语言。

📌 婴儿是真正的”世界公民”。但是，这一项技能并不是永远具备的。和大脑最终会剪切掉没有使用或没有充分使用的突触的情况相似，听和说出任何语言的无限潜能很早就被修剪掉了，只留给我们使用母语的显著能力。

📌 在七个月大的时候，婴儿还属于”世界公民”，他们可以毫不费力地区分出英语中的”r”音和”l”音，但三个月后，这项能力就会消失。

看电视能刺激语言发育吗

大脑并非语言录音机，只有在真实互动中方能真正学会语言。实验显示，仅听真人面对面讲话的婴儿能分辨新语音，而听录音或看视频的婴儿毫无改变。这一发现的核心指向同一结论：社会互动才是大脑发育最关键的推动力。

📌 大脑不是被动地学习语言，而是只在社会回应和社会互动的环境下学习语言。在婴儿与看护人的关系中，语言上进行你来我往的重要性是语言学习过程中的关键因素。在学习过程中，这一因素无比重要。

📌 经过12次实验室观察后，听到真人说话的婴儿能够辨别出汉语普通话的语音，而听到录音或视频设备说话的婴儿则没有任何变化。

何时都不晚

最新的研究带来了一个宽慰的结论：大脑的可塑性并未真正消失，只是被某种”刹车”机制抑制了。这意味着即使错过了关键期，仍有通过后续干预来弥补的空间。为时不晚，机会尚存。

📌 新发现与之前的想法相反，大脑实际上并没有失去其可塑性，而是具备了无限重连的能力。

📌 即使你错过了一些关键阶段，你或许仍可以回到那一阶段去恢复一些东西。‘在以后进行干预，以弥补失去的时间’的想法非常具有吸引力。“

第四章 语言的力量：从思维能力到人生观

父母语言的意义，远不止增加词汇量那么简单。养育者说了多少、如何说，会渗透到数学、空间推理、读写、自控、抗压、毅力乃至道德品质等认知与品格的诸多维度。这种语言滋养无法标价售卖，却是每个孩子最基本、最珍贵的资源——它从根本上塑造了一个人将成为谁、能做什么以及如何去做。

📌 “父母语言”的定义具有迷惑性，因为”父母语言”的神奇作用远不止简单的词汇导入。根据父母对孩子说的词汇量和父母对孩子说话的方式，父母语言会影响孩子数学、空间推理和读写能力的发展，以及孩子约束自身行为和应对压力的能力；且孩子的毅力和道德品质也会受到影响。我认为所有的父母都应该知晓这一点。

📌 看护人的语言无法买卖、存储，也无法在纽约证券交易所上市，但是他们的语言是每个国家、每种文化、每个人的基本资源，甚至扩展到了我们是谁、我们能做什么以及怎么做的方方面面。

神经连接体

所谓”连接体”，就是每个神经元向上万条通路延伸的连接点，它将大脑中上千亿个神经细胞彼此联通，最终决定了我们是谁、我们如何思考和行动。

📌 这样的互连回路被称为连接体。而连接体是每个神经元的上万个连接点，它连接着我们每个人的大脑中的上千亿个神经细胞，是决定我们是谁、我们的思考方式和行为的关键。

父母语言与儿童大脑回路

天赋并非一份固定的礼物，而是一颗种子。它能否真正发芽，全看三岁前浸润在什么样的语言环境里。好的环境能唤醒潜能，贫瘠的环境则会悄悄掩埋它。我们与生俱来的底色，远没有早期经历那双手的塑造力强大。

📌 虽然遗传学的作用不容低估，但科学表明，小孩子早期的语言环境，即父母语言环境很大程度上决定了先天潜能的发挥。

📌 科学告诉我们的是，要发挥出我们的潜力，主要取决于我们出生后三年内大脑发育过程中的经历。简而言之，血统赐予我们的遗传潜力会受到消减和破坏，或者有幸因为我们儿时所经历的父母语言环境而被我们发挥出来。

“我讨厌数学！”

一个例子颇能说明问题：作者的女儿11岁时突然宣称讨厌数学，四年后却成了数学高手。这中间的变化说明，早期无意中传递的性别偏见可能会打击孩子的信心，但只要引导得当，局面完全可以扭转。

📌 “我讨厌数学！”我最大的孩子吉纳维芙在她11岁时如是说道。她激烈地反复强调她不喜欢数学。四年后，她长高了九英寸，并变成了一名数学高手。

📌 发现犯错后我们所采取的一些措施可能很好地改变了女儿并挽救了她糟糕的数学成绩。

数学是通向未来的一扇窗户

PISA测试揭示美国数学成绩落后于多个亚洲地区，前教育部长的首要建议并非改革课程，而是改善从出生到五岁的早期教育。生命最初五年，已经为日后的数学分数埋下了伏笔。

📌 中国上海、中国香港、中国台湾和新加坡在此排名中均名列前茅。据研究表明，中国上海15岁学生的数学成绩”领先美国马萨诸塞州同年龄段学生两年以上”，虽然马萨诸塞州在美国表现强劲。

📌 但他的头等建议是改善从出生至五岁的幼儿所接受的教育。因为出生后的五年对他们今后的学习成绩（包括数学）来说至关重要。

为什么有的人觉得容易，有的人觉得难呢

数学能力并非后天灌输，而是从生命第一天便根植于我们体内。六月龄的婴儿已能将数字发音与实际物体数量对应起来，这种早期的数感，竟能预示其日后数学水平的高低。

📌 研究者们发现数学能力几乎从生命开始的第一天就伴随着我们。

📌 更令人印象深刻的是，婴儿在6个月大时将语音数量和物体数量联系起来的能力通常预示着他最终的数学能力。

略估数字系统

我们生来便配备了一套粗略数量直觉——不必计数，一眼就能感知哪群人数更多。但这种底层数感远不足以托起高等数学的思考。真正的数学思维仍需语言的反复打磨。早期语言环境的分量，在这一刻再度浮现。

📌 不幸的是，虽然我们天生的数感似乎让我们能够正确理解与数字有关的词汇，但是实际上并非如此。事实证明这种数感只是很重要而已。

基本原理：不是简单的”一二三”

入学的数学水平不只是当下的分数，而是一份长达十多年的能力预告。这份起点的差距，主因不在天赋，而在生命头三年浸润了怎样的语言环境。

📌 孩子在入学时的数学技能能够预测他三年级时的数学和读写能力以及15岁时的数学能力。虽然先天的数学潜力可能会发挥作用，但在决定哪些孩子将在入学时具备一定的数学技能方面，前三年语言环境的差异似乎起着重要作用。

父母的数学词汇与儿童的数学能力

不同家庭之间，数学词汇的接触量悬殊得触目惊心——一周内，有的孩子只听到8个，有的却听到1799个。一年累积下来，便是1500个与10万个的鸿沟。听到更多数学词汇的孩子，理解数学基本原理时明显比同龄人更加从容。这道隐形的起跑线，在生命最初几年就已悄然划定。

📌 一周时间内，有的孩子共听到8个数学词汇，而有的孩子则听到了1799个数学词汇。一年之内，听到1500个数学词汇的孩子与听到10万个数学词汇的孩子之间就会存在巨大的差距。

📌 毫无疑问，听到更多数学词汇的孩子选择相应数量点数的卡片的可能性更大。和其他只听到很少数学词汇的孩子相比，这些孩子对数学基本原理的高度理解证实了父母语言的力量。

父母语言影响儿童的空间能力

空间思维能力是预测科学、技术、工程和数学领域成就的关键指标。日常中听到更多“圆形”“方形”“高的”“大的”这类词汇的孩子，空间测试表现明显更优——这与先天聪明无关，直接相关的只是空间词汇的接触量。两岁时掌握的空间词越多，四岁半时的空间推理能力就越强。

📌 空间能力是科学、技术、工科和数学成就的一项重要预测指标，也是父母语言的基础。

📌 莱文教授和她的团队发现，听到和使用更多空间词汇的孩子在空间测试中表现得更出色。数据表明，这不仅仅是因为他们”更聪明”，而是完全与他们所接触到的空间词汇有关。

性别差异：细微的影响如何产生强烈的作用

研究发现，两岁以下女孩听到的数学对话不及男孩的一半，基本数字词汇更是仅及三分之一。这种隐性的性别偏见，到七岁前后会沉淀为自我定型，往往由母亲传给女儿，循环往复。一旦被贴上“不擅长数学”的标签，其数学表现真的会变差，原因并非学不会，而是信念被扼住了。幸而，只要老师自己对数学没有焦虑，课堂里的女生成绩便能与男生持平。

第五章 3T原则：优化大脑发育的亲子沟通

带孩子只需记住三件事。第一，把视线降下去，认真看孩子在关注什么，然后跟着一起投入。第二，把所见所思都用语言摊开，像现场解说般描述正在发生的一切。第三，把互动变成接球游戏，等孩子把话抛回来时稳稳接住，再递回去。这套方法表面平常，却悄然浇灌着数学、读写、自控、情商、创造力和意志力等诸多深层能力。

📌 3T原则即共情关注(Tune in)，充分交流(Talk more)，轮流谈话(Take turns)。在数学概念、读写能力、自我管理、执行力、批判性思维、情商、创造力和毅力培养等方面，3T原则也能发挥相当的引领作用。

第一部分 明确大脑发育的最佳阶段

聪慧并非天生写在基因里，而是从善于沟通的父母那里生长出来。才智是一团可塑的泥土，不是一块刻好的石碑。大脑最终发育成什么样子，取决于是否浸润在丰富的语言环境中——恰如种子能否开出花，全看是否得到恰当的照料。

📌 “3000万词汇倡议”所秉持的基本理念，同时也是科学验证的真理：“没有一个天生聪慧的孩子，他们的聪慧源于后天的培养。“即是说，人的才智具有可塑性。

📌 正如每一颗种子都有潜力成长为玫瑰、牵牛花或绣球花，但最终的结局，还得看它们是否得到了恰当的照料。

3T原则详解：

共情关注（Tune in）： 这是3T原则中最微妙也最基础的一条。其第一步，是看见孩子此刻被什么吸引，而非硬塞过去我们自认为重要的东西。手机、平板最擅长截走这份注意力。唯有心神真正安放在孩子身上时，那些对大脑发育关键的互动才会发生。长期浸润在这种关注中的孩子，注意力更稳，认知和情商也都更出色。

📌 第一个T：共情关注在3T原则中，“共情关注”是最细微的一条。它要求父母有意识地去观察孩子在关注什么。等时机成熟，父母再跟孩子谈论它。换句话说，孩子关注什么，你就关注什么。

📌 数字设备是”共情关注”原则的克星。计算机、平板电脑、智能手机很容易使人上瘾，吸引人的注意力。只有父母把注意力集中到孩子身上时，大脑发育的关注力才会产生。

充分交流（Talk more）： 这一条的目的是让孩子掌握丰富的词汇。一边做事一边把过程说出来，就是在帮孩子建立语音与意义的联结。把焦点放在孩子正在做的事情上，用“车”“球”这类具体名称代替“它”。多聊聊不在眼前的事物，语言便不再局限于此地此刻。孩子说出一个词，我们回以两三个词；孩子说出两三个词，我们回以完整的短句。这样的接力，一次次向前延伸，语言之路便越走越宽。

📌 讲述除了能扩展孩子的词汇量，还能让孩子明确知道词汇发音和意义之间的联系。即听到一个词语，就立马知道它所指的是什么东西或事情。

📌 能够达到后一个阶段的语言水平，恰恰是孩子智力提升的一个重要标志。

轮流谈话（Take turns）： 这是亲子交流的黄金准则，也是3T原则中最关键的一环。我们要做的不是催促，而是学会等待。捕捉到其发出的任何一个声音或眼神信号后，停下来，把说话的下一棒交回去。少问那些一个“是”或“否”就能终结对话的问题，多问“怎么办”“为什么”——这种开放式的接力，才能让思维真正跑起来。

📌 第三条原则叫作”轮流谈话”，它要求父母和孩子在交流时轮流参与谈话。它不仅是亲子交流的黄金准则，还是3T原则中最重要的一环，对于开发儿童的大脑起着决定性的作用。

📌 家长只需将”怎么办”和”为什么”抛给孩子，就足以让他们在思维的世界里天马行空了。

📌 让孩子们自己做主，是培养自制力的重要途径之一。

我们日常说话的风格，正在雕刻截然不同的人生轨迹。一些家庭的对话充满推理、辩论和文字游戏，几乎不用命令；另一些家庭则以服从为先，多用简单指令。这并非爱的深浅，而是语言土壤的质地不同——其深刻影响着孩子的认知根基与学习能力的走向。

📌 中产阶级的父母以一种极其狂热的力量”培养孩子们的天赋、思想和技能”。拉鲁教授称之为”协作培养”。

📌 拉鲁教授将低收入阶层的育儿方式称为”成就自然成长”。这是一种绝对真理就是服从和尊重权威

父母多做一点，孩子更多成长

“协作培养”与”成长型思维模式”有一个共同的核心理念：都认同儿童的智力具有可塑性。相反，如果父母持有固定型思维模式，认为能力天生不变，就更容易做出不利于孩子智力发展的育儿行为。

📌 父母的固定型思维模式将导致不太利于孩子智力发育的育儿行为。

特里西娅的故事

特里西娅自己只读到七年级便辍学，却从未动摇对教育的信念。这个故事让我们看清：成长型思维并非课堂教出来的，而是父母用自身的态度，在日复一日中悄然传给下一代的。

📌 只有尝试才能战胜失败。——特里西娅，波西娅的妈妈

📌 她帮助妈妈们理解她们在孩子们的成长过程中的重要作用，使她们成为孩子们需要的助力者和支持者。

打破固有思维

科学研究早已确认，出生到三岁是大脑发育最关键的窗口，早期语言环境至关重要。可我们即便知晓，行动却总是落在后面——知道和做到的鸿沟，依然横亘在无数家庭面前。

📌 科学是无可争议的。从出生到三岁是开发人的大脑的最关键的几年。这并不意味着你一到四岁大脑就停止发育了，但那几年是最关键的。

科学是真正社会变革的基础

把资源投向孩子生命最早的那几年，回报远比想象中丰厚。赫克曼教授的计算表明，这笔投入能带来每年7%到10%的经济回馈。更重要的是，其学业表现更稳，言行更自律，成年后的生产能力也更扎实。

📌 诺贝尔奖获得者詹姆斯·赫克曼曾说过，我们在高质量的早期教育上为贫穷儿童投资的每一分钱都会得到7%～10%的经济年增长率。

儿童潜能，重中之重

一个国家的重大决策，依赖集体理性的根基。而个人理性思考的能力，其根系在幼年时便已扎下——离不开科学、便利且值得信赖的早期语言环境。

📌 一个国家的至高决议必须建立在可靠的、理性的思考之上，而非感觉。如果一个人想要思考全面、理性的话，那么他的大脑在幼年时期就必须得到充分的发展，而这一切都离不开可靠、科学以及便利的教育。

潜能差距导致成就差距

孩子间的差距，早在出生后九个月便已悄然拉开——细微到肉眼无法察觉，却真实地藏在数据里。找到根源是一回事，找到出路是另一回事，这正是问题棘手之处。

📌 出生后的三年内，从某方面来说，这段时期相对难以看出孩子们之间的差距。实际上，成就差距在孩子出生后九个月便已经存在，只不过相差太小，只能通过数据才能反映出。

父母的语言，第二大资源

父母与孩子的日常交谈，是一座尚未充分开掘的资源。然而真正被用起来的部分，少得惊人。在高质量早期语言环境上每投入一美元，便能带来7%到10%的年回报——改善的不仅是学业与言行，还有成年后的生产能力。

📌 在早期语言环境中，父母和孩子交谈时，说的话的数量和质量是一种极其宝贵、亟待开发的资源。

📌 诺贝尔奖获得者詹姆斯·赫克曼曾说过，我们在高质量的早期教育上为贫穷儿童投资的每一分钱都会得到7%～10%的经济年增长率。同时，这些贫困儿童也会在学业上和言行上有所进步，成年后还会有更高的生产能力。

詹姆斯的故事

詹姆斯从受益者变成了传播者。其用同样的方法养育自己的孩子，还主动把手里的方法教给身边的朋友——不想只让自己孩子拥有这份优势，更愿每个孩子都能获得同样的成长机会。

📌 “为什么我会告诉朋友们？“詹姆斯说道，“我告诉他们，是因为我想让他们的孩子也有我孩子身上的优点。我不想只有马库斯一个孩子知道，或是马库斯因为这一点而占有绝对优势。”

为了孩子的未来

国家潜力若想真正释放，须先让每位公民的潜能充分发挥。在所有资源中，儿童这个最庞大却最被忽视的群体，正是最大的未被开发资源。

📌 一个国家想要释放出自己的全部潜能，必须先将其人民的潜能发挥出来。
